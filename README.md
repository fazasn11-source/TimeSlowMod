# Time Slow Mod ⏰

Мод для Minecraft 1.16.5 Forge, который визуально замедляет время при нажатии клавиши B.

## Особенности
- Нажми **B** чтобы замедлить время в 2 раза
- Повторное нажатие **B** возвращает нормальное время
- Плавные переходы
- Визуальный индикатор на экране
- Звуковые эффекты
- Работает на любых серверах (только визуально)

## Установка
1. Скачай файл `timeslowmod-1.0.0.jar`
2. Положи в папку `mods` Minecraft
3. Запусти игру с Forge 1.16.5

## Как использовать
Просто нажимай **B** в игре!
